#### 기본 데이터 타입은 아니지만 데이터 타입의 위치에서 표현하는 역할을 수행한다.

------------------

## 1. Any : Swift의 모든 타입을 지칭하는 키워드

- Any 타입을 다른 타입에 할당할 수 없다.

------------------


## 2. AnyObject : 모든 클래스 타입을 지칭하는 프로토콜

- 클래스 인스턴스가 아닌 타입 값을 할당할 수 없다.


------------------


## 3. nil : 없음을 의미하는 키워드

- Any가 어떤 데이터 타입이든간에 들어올 수는 있지만, 빈 값은 들어올 수 없다.
